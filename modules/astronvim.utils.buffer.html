Module astronvim.utils.buffer

AstroNvim Buffer Utilities

Buffer management related utility functions

This module can be loaded with local buffer_utils = require "astronvim.utils.buffer"

Functions

is_valid(bufnr)
Check if a buffer is valid

Parameters:

  • bufnr the buffer to check

Returns:

    true if the buffer is valid or false
move(n)
Move the current buffer tab n places in the bufferline

Parameters:

  • n numer of tabs to move the current buffer over by (positive = right, negative = left)
nav(n)
Navigate left and right by n places in the bufferline

Parameters:

  • n the number of tabs to navigate to (positive = right, negative = left)
nav_to(tabnr)
Navigate to a specific buffer by its position in the bufferline

Parameters:

  • tabnr the position of the buffer to navigate to
close(bufnr, force)
Close a given buffer

Parameters:

  • bufnr ? the buffer number to close or the current buffer if not provided
  • force ? whether or not to foce close the buffers or confirm changes (default: false)
close_all(keep_current, force)
Close all buffers

Parameters:

  • keep_current ? whether or not to keep the current buffer (default: false)
  • force ? whether or not to foce close the buffers or confirm changes (default: false)
